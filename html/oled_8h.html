Also has some functions for drawing different shapes on the screen

Goes to the position (row, column) on the screen that we want to change.

Moves the pointer to the row containing 8 vertical bits and column with 1 horizontal bit that we wants to change. To change the uint8_t bit on this position we write to the address to oled_data. 1 means pixel ON and 0 means pixel OFF.

Parameters
